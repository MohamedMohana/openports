# OpenPorts Homebrew Tap

This tap provides a Homebrew cask formula for [OpenPorts](https://github.com/MohamedMohana/openports), a lightweight macOS menu bar app for monitoring local ports and processes.

## Installation

```bash
brew install --cask MohamedMohana/tap/openports
```

## Updating

```bash
brew upgrade --cask MohamedMohana/tap/openports
```

## Uninstallation

```bash
brew uninstall --cask MohamedMohana/tap/openports
```

## Documentation

For full documentation, visit [https://github.com/MohamedMohana/openports](https://github.com/MohamedMohana/openports).
